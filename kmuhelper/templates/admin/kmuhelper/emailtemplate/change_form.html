    Platzhalter

    Wozu Platzhalter

    Mit Platzhaltern können ganz einfach bestimmte Werte einer Bestellung oder eines Kunden direkt in einer Vorlage verwendet werden. Um diese Werte zu verwenden kann auf der entsprechenden Seite der Bestellung oder des Kunden auf "Zu E-Mail-Vorlage" geklickt werden. Die Werte werden dann zwischengespeichert und bei der nächsten Verwendung einer Vorlage automatisch eingesetzt.

    Verwendung von Platzhaltern

    Um einen Platzhalter zu verwenden, kann '@VARIABLENNAME@' in diese Vorlage eingesetzt werden. Platzhalter können in allen Feldern der Kategorie 'Inhalt' verwendet werden. Der Variablenname muss in GROSSBUCHSTABEN geschrieben werden.

    Beispiele von Platzhaltern

    Folgendes Beispiel ist für eine Bestellung:

    Feld Inhalt
    Empfänger @RECHNUNGSADRESSE_EMAIL@
    Betreff Ihre Bestellung #@ID@ wurde versendet
    Text Guten Tag

    Ihre Bestellung mit der ID @ID@ wurde versendet und sollte in den nächsten Tagen eintreffen.
    Sie können die Bestellung unter folgendem Link verfolgen: https://www.post.ch/swisspost-tracking?formattedParcelCodes=@TRACKINGNUMMER@
